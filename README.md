# SplatStudio

A toddler-friendly drawing app built with TypeScript, Vite, and Bun.

## Prerequisites

This project uses [Bun](https://bun.com) as its JavaScript runtime.

### Install with mise

If you use [mise](https://mise.jdx.dev) to manage tool versions:

```bash
mise install
```

This will install the correct version of Bun as defined in `.mise.toml`.

### Install manually

```bash
curl -fsSL https://bun.sh/install | bash
```

## Getting started

```bash
# Install dependencies
bun install

# Run the dev server
bun run dev
```

Then open http://localhost:5173 in your browser.
